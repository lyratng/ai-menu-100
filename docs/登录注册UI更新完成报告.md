## 🎯 核心设计元素

### 1. 色彩系统
```css
背景色: #F5F5F0  /* 温暖米白色 */
主色:   #2C2C2C  /* 深灰色 */
边框色: #E8E8E3  /* 浅灰米色 */
文字色: #999     /* 中灰色 - 次要文字 */
        #666     /* 深灰色 - 描述文字 */
```

### 2. 字体系统
- **字重**: `font-light` (300) - 轻盈优雅
- **字间距**: `tracking-wide` - 提升可读性
- **Logo**: `tracking-[0.1em]` - 增强品牌感

### 3. 按钮风格
```css
默认状态: bg-[#E8E8E3] text-[#999]
Hover状态: bg-[#2C2C2C] text-white
过渡动画: transition-all duration-300
```

### 4. 输入框
```css
高度: h-12 (登录/注册主页) / h-11 (注册子页面)
边框: border-[#E8E8E3]
焦点: focus:border-[#2C2C2C]
```

## 🔍 细节优化

### 1. 字间距 (Letter Spacing)
- Logo: `tracking-[0.1em]` (超宽)
- 标题: `tracking-wide` (宽)
- 正文: `tracking-wide` (宽)

### 2. 圆角 (Border Radius)
- 卡片: `rounded-lg` (8px)
- 输入框: `rounded-md` (6px)
- 按钮: `rounded-md` (6px)

### 3. 阴影 (Box Shadow)
- 卡片: `shadow-lg` (大阴影)
- Hover: 轻微加深

### 4. 过渡动画
- 所有交互: `transition-all duration-300`
- 颜色变化: `transition-colors`
